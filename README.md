# NeuroBloom – Cognitive Fitness Analyzer

NeuroBloom is an AI-powered cognitive fitness analysis system that evaluates user stress levels and focus scores to generate adaptive scores and personalized recommendations.

## 🚀 Features
- Real-time stress & focus analysis
- Adaptive score calculation
- Personalized recommendations
- Interactive data visualization using charts

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript, Chart.js
- Backend: FastAPI (Python)
- Database: SQLite

## ⚙️ How It Works
Users enter stress level and focus score. The backend processes this data using an adaptive algorithm and returns insights, which are displayed along with charts on the frontend.

## 🎯 Outcome
Developed a full-stack AI-integrated web application demonstrating real-world problem solving in cognitive health analysis.
